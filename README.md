## Sistema de controle de estoque

Aplicação para gerenciamento de estoque, desenvolvida para simular o funcionamento de um sistema real utilizado por empresas para controle de produtos, categorias e movimentações de entrada e saída.
O sistema permite acompanhar o nível de estoque, registrar movimentações e gerar informações importantes para a tomada de decisão, aplicando regras de negócio comuns no mercado.

O objetivo desse projeto é consolidar o conhecimento em APIS, CRUD, integração entre backend, frontend e banco de dados.

## Tecnologias a serem utilizadas

**Backend**: Java, SpringBoot, Spring Data JPA, Hibernate e API REST. <br>
**Frontend**: HTML, CSS e JavaScript. <br>
**Banco de dados**: PostgreSQL
